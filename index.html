IRMHUB

Universal PowerShell Tool Launcher
One command to rule them all.

Launch IRMHUB

Open PowerShell (or Windows Terminal) and run:

PS >irm https://raw.githubusercontent.com/MYMDO/irmhub/main/irmhub.ps1 | iex
Security reminder: Always verify the URL before running any irm | iex command. Inspect the raw script at the link above before executing. IRMHUB shows you every command before running it and requires manual confirmation.

Security Model & Privacy

TLS 1.2+ enforced dynamically
Zero telemetry, no outbound logs
Runs in isolated child scope
Requires manual typing of "YES"
